# Ballots from the 2007 Orsay experiment

## Correspondance number-mention

In the CSV files, mentions are encoded by numbers as follows:

| Number | Mention |
|---|---|
| 6 | Très bien |
| 5 | Bien |
| 4 | Assez bien |
| 3 | Passable |
| 2 | Insuffisant |
| 1 | À rejeter |
| 0 | Sans opinion / blanc |
